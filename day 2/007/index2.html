Thẻ input có type text
        có các thuộc tính type ( kiểu của ô input )
        thuộc tính name ( tên của ô input ) viết liên và không dấu
        thuộc tính value="" ( giá trị của input )
        placeholder ( hiện ra 1 văn bản để gợi ý cho người dùng biết cần nhập cái
        gì vào ô input và nó chỉ hiện khi ô input rỗng )
