Thuộc tính id của thẻ html
        thuộc tính id được sử dụng để định đanh cho 1 thẻ html là duy nhất
        không được sử dụng phép có 2 thẻ có id trùng nhau trên 1trang web
        hiểu đơn giản là id như số chứng minh thư

        thuộc tính id có ý nghĩa quan trọng css và javascrip

        Trong css sẽ có 1 khái niệm là bộ chọn ( selector )
        Bộ chọn này khi muốn chọn 1 thẻ html dựa vào thuộc tính id của html
        thì sẽ viết là #id
        ví dụ trong file này :
        #target1
        #target2
        #target3
Thẻ id số 1
Thẻ id số 2
Thẻ id số 3
