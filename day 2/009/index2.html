thuộc tính class trong html
        thuộc tính class được sử dụng để gom các thẻ html vào nhóm theo y muốn
        thẻ html có thể chung class
        khi mà có nhiều class trong 1 thẻ thì viết cach nhau ra
        ví dụ:
            class ="class1 class2 "
        trong css muốn chọn được các thẻ dựa trên class  thì sử dụng dấu .
        ví du .miên bac hay .trunguong
