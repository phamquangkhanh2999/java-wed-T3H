        thẻ ol dạng đánh số từ 1 đến n

        list-style-type
        giá trị :
        circle ( hình tròn )
        square ( hình vuông )
        upper-roman ( số la mã )
        lower-alpha ( số abc đến z )
        none ( không có gì )
