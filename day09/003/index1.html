.col- (extra small devices - screen width less than 576px)
        => áp dung màn hình điện thoại
        .col-sm- (small devices - screen width equal to or greater than 576px)
        => áo dụng màn hình ipad mini ( máy tính bảng )
        .col-md- (medium devices - screen width equal to or greater than 768px)
        => áp dụng màn hình laptop
        .col-lg- (large devices - screen width equal to or greater than 992px)
        => màn hình to máy tính bàn
        .col-xl- (xlarge devices - screen width equal to or greater than 1200px)
        => màn hình siêu lớn ( tivi )
    
         Khung chứa bên ngoài cùng ( class .container )
        Trong khung chứa có các hàng ( class .row )
        Trong cùng các cột ( hàng chứa cột ) cột có tiền tố là col-*
        * chính là 1 số từ 1 đến 12
        Bootstrap sử dựng lưới 12 cột
    
        công thức => ? = 12/số_cột
        1 cột => .col-sm-? => .col-sm-12
        2 cột bằng nhau => .col-sm-? => .col-sm-6
        3 cột bằng nhau => .col-sm-? => .col-sm-4
        4 cột bằng nhau => .col-sm-? => .col-sm-3
        6 cột bằng nhau => .col-sm-? => .col-sm-2
        12 cột bằng nhau => .col-sm-? => .col-sm-1
